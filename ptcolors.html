messages(info_msg, success_msg, failure_msg, callback, *args, **kwargs)¶

Context manager that prints messages and runs a callback.

Parameters:

  • info_msg (str) – Info message to be printed on entry.

  • success_msg (str) – Success message to be printed on exit if the callback succeeds.

  • failure_msg (str) – Failure message to be printed on exit if the callback fails.

  • callback (callable) – Callback function to be run in the context.

  • args – Positional arguments for the callback.

  • kwargs – Keyword arguments with expected Exception.

To use this context manager, you can do the following:

with PTColors().messages(
    "Starting...",
    "Success...",
    "Failure...",
    callback,
    *args,
    **kwargs,
) as status:
    self.status = status
